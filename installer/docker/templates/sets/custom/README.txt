Custom email template set

How to use:
1) Edit the HTML/ERB files in this folder.
2) Run: scripts/preview-chatwoot-templates.sh custom
3) In whatpro-setup, choose:
   - 16) Select email template set -> custom
   - 14) Chatwoot email templates

Notes:
- Use the existing sets as reference.
- Keep filenames the same as Chatwoot expects.
